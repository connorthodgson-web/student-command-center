# Student Command Center

Student Command Center is an AI-powered student assistant designed to help students talk through their school responsibilities, quickly capture tasks in natural language, and stay supported throughout their academic life.

## Vision

School life is messy. Deadlines live in many places: class portals, notes apps, paper handouts, group chats, memory, and last-minute reminders from teachers or friends.

Student Command Center is meant to feel like a personal academic assistant that helps a student make sense of all of that.

The product should let a student:
- say what they need to do in normal language
- ask questions about their workload
- keep track of classes, assignments, and tests
- receive reminders and summaries
- build a personalized school support system over time

This is not just a planner and not just a chatbot.
It should feel like an assistant first.

## MVP Priorities

The first version of the product should focus on:

- class schedule setup
- natural-language task capture
- AI chat over student workload
- reminder settings and reminder architecture
- a clean, simple student dashboard
- strong foundational data models for future personalization

## Example natural-language inputs

- "I have a Great Gatsby essay due March 11 at 11:59 PM."
- "Math homework due tomorrow."
- "Bio test next Friday."
- "Remind me to study chemistry tonight."
- "History reading due Monday before class."
- "I need to finish my lab write-up this weekend."

## Product Philosophy

Student Command Center should feel like:
- an academic executive assistant
- conversational
- helpful
- calm
- personalized
- easy to use

It should not feel like:
- a generic to-do app
- a rigid school tracker
- a dashboard with random AI features attached
- a fake urgency engine pretending to know the student's life better than they do

For the MVP, the focus is not advanced urgency ranking.
The focus is assistant usefulness.

## Early Non-Goals

The following are intentionally out of scope for the first version:

- Canvas integration
- advanced tutoring
- syllabus parsing
- web research for every subject
- advanced autonomy or agent actions
- mobile apps
- parent dashboards
- fully automated academic planning

## Tech Direction

Planned stack:
- Next.js
- TypeScript
- Tailwind
- Supabase
- OpenAI API
- Vercel

## Development Principle

Build the assistant experience first.
The interface should support the assistant, not replace it.
