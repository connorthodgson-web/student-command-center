# Current Sprint

## Sprint Goal

Create the first working scaffold for Student Command Center as an assistant-first student app.

## Current Focus

Build the foundations for:
- class setup
- natural-language task capture
- AI chat structure
- reminder settings structure
- a clean dashboard that supports the assistant experience

## Definition of Done

This sprint is successful when:
- the project structure exists
- the key docs exist
- the main pages exist
- reusable starter components exist
- type definitions exist
- mock data exists
- placeholder API routes exist
- the codebase clearly reflects the assistant-first product direction

## Explicit Non-Goals

Do not build yet:
- Canvas integration
- advanced tutoring
- syllabus parsing
- advanced urgency ranking
- background scheduling infrastructure
- production-ready auth
- final polished UI

## Next Likely Sprint

After this scaffold, the next sprint should likely focus on:
- natural-language task parsing flow
- task creation and storage
- initial chat behavior
- reminder preference persistence
