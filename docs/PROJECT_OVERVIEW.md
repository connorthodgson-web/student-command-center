# Project Overview

## Project Name
Student Command Center

## Core Idea

Student Command Center is an AI-powered student personal assistant.

Its purpose is to help students manage school by letting them interact naturally with an assistant that understands their classes, tasks, reminders, and workload.

The student should be able to talk to the app the way they would talk to a helpful academic assistant:
- "I have a paper due Friday."
- "What do I need to do tonight?"
- "When is my next test?"
- "Remind me to study after dinner."
- "What do I still have left for English?"

## Core Promise

Turn messy school life into clear, personalized support.

## MVP Focus

The MVP should prioritize these capabilities:

1. Class and schedule structure
2. Natural-language task capture
3. AI chat over student data
4. Reminder preferences and reminder architecture
5. A clean dashboard that supports the assistant experience

## Product Identity

This project should be built as an assistant-first product, not a planner-first product.

That means:
- conversation matters more than complex dashboards
- natural-language input matters more than manual form filling
- personalization matters more than automatic productivity scoring
- flexibility matters more than pretending the system knows the perfect urgency of every task

## Key Design Principles

- Assistant first
- Clean and calm interface
- Simple inputs
- Strong structure underneath
- Beginner-friendly codebase
- Small MVP scope
- Expandable over time

## Example User Experience

A student opens the app and sees:
- a calm overview of their school life
- a place to type what they need to do
- a chat interface for asking questions
- a list of classes and upcoming items
- reminder settings

The app should feel supportive, not overwhelming.

## Not in MVP

Do not build these yet:
- Canvas integration
- full tutoring system
- syllabus intelligence
- advanced urgency scoring
- automatic study planning across all courses
- mobile app
- parent or teacher dashboards
