# Roadmap

## Guiding Strategy

Build the student assistant experience first.

The goal is not to make the most feature-heavy student planner.
The goal is to make the most useful AI-powered academic assistant.

## Phase 1 - Assistant MVP

Focus:
- class setup
- natural-language task capture
- task storage
- AI chat over workload
- reminder settings
- clean assistant-oriented dashboard

Success criteria:
- a student can create classes
- a student can enter tasks naturally
- the system stores structured task data
- the student can ask workload questions in chat
- the student can configure reminder preferences
- the app feels like the beginning of a personal assistant

## Phase 2 - Better Personalization

Focus:
- stronger class-task linking
- better natural-language extraction
- conversational follow-up handling
- custom reminders from chat
- better summaries
- more personalized assistant responses

Success criteria:
- the app remembers more context
- the assistant feels more personalized
- reminder behavior becomes more useful
- chat becomes central to how the student manages school

## Phase 3 - Academic Support Expansion

Focus:
- syllabus ingestion
- course context
- test and project planning
- study plan assistance
- tutoring support
- document-aware help

Success criteria:
- the assistant can support more than just task capture
- the assistant becomes a broader academic support tool
- the student can ask deeper course-related questions

## Phase 4 - Product and Business Expansion

Focus:
- onboarding refinement
- landing page
- analytics
- billing
- deployment hardening
- user feedback loops
- retention systems

Success criteria:
- the product is usable by real early users
- onboarding is clear
- the value proposition is obvious
- the app has a path toward business viability

## Notes on Prioritization

For now, do not prioritize:
- complex urgency engines
- flashy productivity scoring
- unnecessary UI complexity

Prioritize:
- assistant usefulness
- smooth natural-language input
- student trust
- personalization
- reminders
- clarity
